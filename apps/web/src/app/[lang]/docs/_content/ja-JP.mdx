# プロットウィストについて

**プロットウィスト（Plotwist）** は、すべてをひとつのプラットフォームにまとめるために作られました。Letterboxd や MyAnimeList のように特定のコンテンツだけに焦点を当てたプラットフォームとは異なり、プロットウィストは映画、ドラマ、アニメ、韓国ドラマなど、さまざまなコンテンツを管理・評価できます。

私たちの目標はシンプルです。リストを整理し、視聴履歴を追跡し、視聴予定を計画するための完全で洗練されたスペースを提供することです — すべてがひとつの場所に。

## 無料プランの機能

- **プラットフォームへのフルアクセス**  
  エンターテインメントを管理するための主要なツールを探索して活用しましょう。  

- **無制限の評価**  
  制限なく、あらゆるタイトルについての感想を共有できます。  

- **無制限のリスト作成**  
  好きなだけコンテンツをリスト化して整理しましょう。  

- **コンテンツ管理**  
  視聴したもの、視聴中のもの、視聴予定のものを簡単に追跡できる便利なツールです。  

- **共同リスト**  
  友達とリストを作成し、インタラクティブに提案を共有できます。  

- **友達へのコンテンツ推薦**  
  プラットフォーム内で直接友達に推薦を送信できます。  

## プロプランの機能

> **プロプラン** のいくつかの機能は現在開発中で、近日中に公開予定です。

**プロプラン** を購読することで、さらにパーソナライズされた充実した体験を提供する高度な機能を利用できます：

- **履歴に基づく個別推薦**  
  あなたの好みにぴったり合った新しいコンテンツを発見できます。  

- **バッジと実績**  
  独自のバッジを使って進捗を示し、特別な報酬を解除できます。  

- **詳細な個人統計**  
  最も多く視聴したジャンルや総視聴時間など、視聴習慣に関する詳細な情報を確認できます。  

- **新機能への優先アクセス**  
  プラットフォームの新しい機能をいち早く試せます。  

## TMDBとの統合

TMDB との統合により、**プロットウィスト** は映画、ドラマなどの膨大なカタログを提供します。あらすじ、キャスト、予告編、その他の詳細情報が揃っています。
